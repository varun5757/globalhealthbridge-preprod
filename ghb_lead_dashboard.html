📊 Google Sheets — Backend Setup

⏱ One-time setup · Takes about 10 minutes · Completely free
1
Create a new Google Sheet
Go to sheets.google.com → click "Blank" → rename it to "GHB Leads" (click the title at top left).
2
Open Apps Script
In your Google Sheet → click Extensions menu → Apps Script. A new tab will open with a code editor.
3
Paste the backend code
Delete everything in the editor. Copy the code from the file google_apps_script.js (provided alongside this dashboard) and paste it in. Click the 💾 Save icon.
4
Deploy as Web App
Click Deploy → New deployment. Set type to "Web app". Set "Execute as" → Me. Set "Who has access" → Anyone. Click Deploy → Authorise when prompted → Copy the Web App URL.
5
Paste the URL below and save
Paste your Web App URL in the field below. Click Save & Test — the dashboard will connect to your Sheet instantly.

📱 WhatsApp Alerts via CallMeBot

⏱ Takes 2 minutes · Free · No app needed
1
Save this number in your phone contacts
Add +34 644 52 71 67 as a contact named "CallMeBot".
2
Send this exact WhatsApp message to that number
I allow callmebot to send me messages
3
You'll receive your API key in 1–2 minutes
CallMeBot will reply with your personal API key. Copy it and paste below.
4
Add your API key to the Google Apps Script
In your Apps Script file, find line: const CALLMEBOT_API_KEY = 'YOUR_CALLMEBOT_API_KEY';
Replace YOUR_CALLMEBOT_API_KEY with your actual key. Click Save → then Deploy → New deployment again. Use the new URL.

📧 Email Alerts via EmailJS

⏱ Takes 5 minutes · Free up to 200 emails/month
1
Sign up at EmailJS
Go to emailjs.com → Sign Up Free → use your Gmail to register.
2
Add Gmail as an Email Service
Dashboard → Email Services → Add New Service → choose Gmail → connect globalhealthbridge.in@gmail.com → click Create Service → copy your Service ID.
3
Create an Email Template
Email Templates → Create New Template
Subject: New Lead: {{lead_name}} — {{lead_specialty}}
Body: include these variables:
{{lead_name}} · {{lead_email}} · {{lead_phone}} · {{lead_country}} · {{lead_specialty}} · {{lead_description}} · {{lead_date}}
Click Save → copy your Template ID.
4
Get your Public Key
Click your account name (top right) → Account → copy the Public Key.
5
Paste all 3 keys below
Fill in the fields below, then click Save & Test. You'll receive a test email at your Gmail.
